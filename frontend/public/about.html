Busy Cafe는 이렇게 계산해요

카페 안의 좌석이나 대기 인원을 측정하지 않습니다. 서울시가 제공하는 지역 관측을 카페 위치와 연결해 주변 지역이 상대적으로 얼마나 붐비는지 추정합니다.

마커 색은 매장 내부 혼잡도가 아닙니다. 방문 전 관측 지역, 거리, 데이터 시각과 외부 지도 장소 정보를 함께 확인해 주세요.

최근 업데이트

  • 서울 지하철 19개 노선, 380개 역 지점과 2,126개 출구를 지도 맥락으로 추가했습니다. 확대하면 출구 번호를 노란 배지로 확인할 수 있습니다. 지하철 정보는 길 찾기를 돕는 표시이며 혼잡도 계산에는 사용하지 않습니다.
  • 화면 상단, 지도 안내와 카페 상세가 휴대전화의 노치·둥근 모서리·하단 홈 영역을 침범하지 않도록 안전 여백을 반영했습니다.
  • 카페 상세에서는 관측 시각·근거 강도 같은 정보를 짧게 묶어 보여줍니다. 모바일 상세 화면은 핵심 정보를 먼저 보여주며, 위쪽 손잡이로 전체 내용을 펼치거나 다시 접을 수 있습니다. 지금 상태 알려주기를 누르면 피드백 양식도 별도로 펼쳐집니다.
  • 출퇴근시간 안내는 현재 방문에서 닫을 수 있습니다. 이 안내는 정해진 시간대의 주의 문구이며, 해당 카페의 정확도를 따로 측정했다는 뜻은 아닙니다.
  • 지도가 갱신될 때 불완전하거나 더 오래된 요약이 먼저 확인한 최신 관측 지역·거리· 시각을 덮어쓰지 않도록 상세 갱신 동작을 수정했습니다.

지도에서 보는 법

주변 혼잡도

여유·보통·약간 붐빔·붐빔의 네 단계로 표시합니다. 같은 지역 안에서 비교하는 참고값입니다.

관측 거리

가까운 서울시 관측 지점 최대 3곳을 거리 가중 방식으로 연결합니다. 가장 가까운 지점과 거리를 함께 표시합니다.

경계 지역

가장 가까운 관측 지점이 600m를 넘으면 경계 지역입니다. 1,500m를 넘으면 현재 혼잡도를 표시하지 않습니다.

데이터 시각

25분 이내는 현재 참고값, 25~120분은 지연 참고값입니다. 120분을 넘으면 현재값을 숨깁니다.

카페 정보

카페 목록은 서버에 미리 저장해 빠르게 제공합니다. 서울에서는 정기적으로 완전 수집한 카카오 로컬 카페 목록을 우선 사용하고, Overture Places와 서울시 휴게음식점 인허가 정보로 누락과 출처를 보완합니다. 외부 지도 버튼은 검증된 상세 링크를 우선 사용하며, 네이버 장소 ID가 없으면 카페명과 주소를 넣은 검색 링크임을 별도로 표시합니다.

장소 원장의 숫자형 품질값은 매장 존재 여부나 혼잡도 정확도의 확률이 아니므로 상세 패널에 노출하지 않습니다. 원본 release와 검증 metadata는 서버에 보존합니다.

데이터 출처와 이용조건

  • 혼잡도: 서울특별시 서울시 실시간 도시데이터(OA-21285) · 공공누리 제1유형
  • 영업 인허가 대조: 서울시 휴게음식점 인허가 정보(OA-16095) · 공공누리 제1유형
  • 장소 fallback: Overture Places · 현행 서울 원장 공급자에 CDLA-Permissive-2.0과 CC0-1.0 적용
  • 서울 장소 우선 원장: 카카오 로컬 API · Kakao API 운영정책
  • 지하철 맥락: 서울시 역사마스터 정보(OA-21232) · 노선 형상과 출구 위치는 OpenStreetMap contributors, ODbL
  • 베이스맵: OpenFreeMap © OpenMapTiles · OpenStreetMap contributors, ODbL

현재 적용 release와 기계 판독 가능한 전체 링크는 전체 출처 목록에서 확인할 수 있습니다. Busy Cafe 코드는 GitHub에서 Apache License 2.0으로 공개하며, 외부 데이터에는 각 제공자의 조건이 별도로 적용됩니다.

운영과 문의

장소 누락은 검색 결과의 신고 양식, 폐업·위치 오류는 카페 상세의 장소 신고 양식으로 알려주세요. 신고는 검증 대기열에만 저장되고 카페 원장을 자동 수정하지 않습니다. 주변 거리 혼잡과 매장 좌석 피드백도 서로 다른 값으로 저장하며, 검증 전 공개 점수나 추천에 자동 반영하지 않습니다. 추가 문의는 GitHub 이슈를 이용해 주세요. 개인정보와 분석 방식은 개인정보·운영 안내에서 확인할 수 있습니다.
